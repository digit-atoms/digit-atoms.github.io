All In One Merge

All In One Merge e' il tool adatto a velocizzare il tuo lavoro.

Basato su keywords

Permette di sostituire le keywords in template spreadsheet e docs.

Generazione PDF

Permette di generare dai template i PDF

Email automatiche

Permette di generare una mail dal template con allegati i PDF

Veloce e Semplice

Ti basta inserire l'id e premere il tasto "MERGE + EMAIL" per fare la sostituzione delle chiavi. 
Ti basta premere "CARICA DATI" per caricare nello spreadsheet attuale i valori con le stesse chiavi da altri sheet

I tuoi dati sono al sicuro.


Quali informazioni raccogliamo: 

Quando utilizzi lo script,richiediamo l’acesso ai tuoi servizi google. Ad esempio, il drive, le
email e i servizi che vedi richiesti nella pagina di consenso oauth.

Non salviamo da nessuna parte i tuoi dati. Verranno utilizzati SOLO per far eseguire lo script
correttamente.

Cosa non raccogliamo

Non abbiamo accesso al tuo numero di telefono oppure alla tua posizione. Se hai dei dubbi
su un tipo specifico di dato personale, puoi contattarci tramite email.

Per cosa utilizziamo i dati
Utilizziamo i tuoi dati per:
● Creare una copia dei template google docs
● Sostituire le parole chiave dal file sheet al file copiato docs
● generare un pdf dal file google docs copiato
● inviare una mail (con il testo generato da un file docs, con le parole chiave sostituite)
allegando i pdf generati
